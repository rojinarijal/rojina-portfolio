# Rojina Portfolio

Welcome to Rojina's Portfolio! This is a full-stack web application showcasing personal projects, skills, and contact information.

## Project Structure

This project is separated into two main parts:

- **Frontend**: A React application powered by Vite, containing the website's design and user interface.
- **Backend**: An Express.js server using SQLite to handle contact form submissions and database logic.

## How to Run Locally

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### 1. Start the Backend Server
Open a terminal in the root directory and run the following commands:
```bash
cd "Rojina page"
npm install
npm start
```
The backend server will start running on `http://localhost:5000`.

### 2. Start the Frontend Website
Open a second terminal in the root directory and run the following commands:
```bash
cd "Rojina page/Rojina page/frontend"
npm install
npm run dev
```
The frontend website will start running on `http://localhost:5173`. You can open this link in your web browser to view the portfolio.

## Technologies Used
- **Frontend**: React, Vite, CSS
- **Backend**: Node.js, Express.js, SQLite3
- **Tools**: npm, Git
